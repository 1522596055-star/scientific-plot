# heatmap

Heatmap-style samples.

## What belongs here
- dense matrix heatmaps
- correlation matrices
- grid-valued color maps

## When to search this category first
- when the data is naturally a 2D matrix
- when color intensity should encode value over row/column combinations
- when row and column labels are both meaningful parts of the figure

## When to avoid this category
- when the data is a list of independent observations rather than a matrix
- when point locations matter and scatter would be clearer
- when category comparisons are better represented with bars
