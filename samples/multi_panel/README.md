# multi_panel

Multi-panel samples with two or more coordinated subplots.

This category is intentionally curated into a **small set of canonical starters** plus a few variants, so agents can learn the main reusable layouts without browsing many similar papers.

## What belongs here
- 1x2, 1x3, 2x2, 2x3, or larger subplot layouts
- repeated-axis comparisons
- per-panel condition or species comparisons

## Read this before browsing too many samples
- `../../patterns/multi_panel.md`

## When to search this category first
- when the figure is naturally divided into several aligned panels
- when one panel per species, condition, or metric is clearer than overlaying everything
- when the same visual grammar repeats across panels

## Canonical starter templates
- `sample_0014` — primary starter for vertically stacked metrics sharing one x-axis
- `sample_0003` — primary starter for dense grid-based validation layouts with model lines and reference markers
- `sample_0007` — primary starter for two-panel validation pairs

## Secondary variants
- `sample_0002` — compact repeated 1x3 validation comparison
- `sample_0010` — shared-x two-panel metric comparison with baseline/reference-line flavor

## How to choose within this category
- start with `sample_0014` for two metrics stacked over a common x-axis
- switch to `sample_0003` when many species/targets need their own coordinated panel
- switch to `sample_0007` when two panels each compare one modeled quantity against a reference
- only open `sample_0002` or `sample_0010` if those narrower structures are explicitly useful

## When to avoid this category
- when a single axis can communicate the result clearly
- when the figure is really just one chart with an inset
- when the main task is a pure distribution chart or heatmap
