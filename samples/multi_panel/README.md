# multi_panel

Multi-panel samples with two or more coordinated subplots.

## What belongs here
- 1x2, 1x3, 2x2, 2x3, or larger subplot layouts
- repeated-axis comparisons
- per-panel condition or species comparisons

## When to search this category first
- when the figure is naturally divided into several aligned panels
- when one panel per species, condition, or metric is clearer than overlaying everything
- when the same visual grammar repeats across panels

## When to avoid this category
- when a single axis can communicate the result clearly
- when the figure is really just one chart with an inset
- when the main task is a pure distribution chart or heatmap
