# bar

Bar-style samples.

## What belongs here
- grouped bar charts
- horizontal bar charts
- sensitivity-style bar figures

## When to search this category first
- when the primary comparison is across discrete categories
- when values are already aggregated per category
- when signed importance values should be shown around zero with bars

## When to avoid this category
- when the data is continuous over x and should be shown as curves
- when the real goal is to show full distributions instead of one value per category
- when the data is matrix-shaped and should become a heatmap

- `sample_0001` — best starting point for grouped vertical bars
- `sample_0008` — best starting point for horizontal signed sensitivity bars
