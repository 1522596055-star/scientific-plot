# scatter

Scatter-style samples.

## What belongs here
- sparse scientific scatter plots
- log-scale scatter comparisons
- bubble scatter / embedding plots

## When to search this category first
- when individual observations matter more than smooth fitted curves
- when groups are represented by marker color, shape, or size
- when the figure should emphasize sparse measured points

## When to avoid this category
- when the target figure needs continuous curves connecting points
- when the real objective is a distribution summary like a boxplot or histogram
- when the point set is actually a dense matrix and should be a heatmap
