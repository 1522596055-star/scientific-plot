# reference_0001

## Why keep this reference
This figure is worth keeping because it shows a **non-trivial dual-axis scientific line layout** that is still calmer than many paper figures of the same type. It is useful when a simple plain multi-line starter is not enough, but a full new executable sample would be misleading without the original data.

## Borrow from it
- the stacked comparison of two operating points
- the restrained dual-axis treatment
- the shared stoichiometric marker as a structural cue

## Do not copy literally
- the exact species / HRR / temperature combination
- the full paper legend burden if your figure can stay single-axis
- the figure if the secondary axis is only decorative
