# reference_0002

## Why keep this reference
This figure is worth keeping because it shows a **dense conditional-profile grid** that would be costly to fake well without the original underlying data. It is especially useful for simulation-heavy work where the figure needs to show both trend and spread across several variables.

## Borrow from it
- the two-case column split
- the row-wise repetition of one visual grammar
- the way spread information stays visible without overwhelming the means

## Do not copy literally
- the exact six-panel quantity set
- the full density if your comparison only needs one or two rows
- the figure if a simpler validation pair or compact repeat is enough
