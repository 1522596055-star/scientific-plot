# reference-only multi-panel figures

Use this category when a panel layout is valuable as a visual reference but cannot be rebuilt cleanly with the available data.

These references usually help with:
- dense validation grids
- repeated panel rhythm
- mean / standard-deviation presentation
- deciding when a paper figure is too specific to promote into `samples/`
