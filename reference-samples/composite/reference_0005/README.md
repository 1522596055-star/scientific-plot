# reference_0005

## Why keep this reference
This figure is worth keeping because it shows how a paper can combine **repeated quantitative summaries with one compressed field strip** without turning the page into a collage. It is especially useful for simulation work where a set of downstream summary curves benefits from one spatial reminder.

## Borrow from it
- the repeated summary-chart rhythm across several cases
- the compressed field strip as context rather than as the main event
- the idea that one field panel can justify several aligned summary charts

## Do not copy literally
- the exact blend count and evaporation-specific variable choices
- the figure if the field strip is only decorative
- the full density when your story needs only one case or one chart block
