# reference_0004

## Why keep this reference
This figure is worth keeping because it shows a compact **field-plus-profile** composition that is highly relevant to simulation-heavy combustion work. The image makes spatial combustion structure legible first, then uses a small profile row to quantify differences between cases.

## Borrow from it
- the clear top-versus-bottom hierarchy
- the idea that extracted profiles should support, not replace, the contour view
- the tight comparison between two cases without needing a large grid

## Do not copy literally
- the exact case labels, fuel chemistry, or combustion-mode notation
- the figure if the field and the extracted cuts are not sampling the same story
- the structure when a simpler two-panel validation pair is enough
