# pdf inbox

Newly downloaded PDFs land here before full triage.

These files are **not** the first retrieval surface for agents. The expected flow is:

1. archive the raw download here
2. inspect the paper
3. decide whether it should become:
   - a new canonical sample
   - a secondary variant sample
   - a distilled note in `patterns/`
   - or just a kept archive with no promotion

Until triage is done, keep the original downloaded filename here so nothing is lost.

For simulation-heavy curation, also consult:
- `../SIMULATION_CURATION.md`
- `NUMERICAL_PRIORITY.md`

See also:
- `TRIAGE.md` — first-pass placement notes and promotion recommendations
- `NUMERICAL_PRIORITY.md` — shorter simulation-weighted shortlist
