# scientific-plotter

Bundled agent skill for turning raw scientific data or vague plotting requests into publication-style figures.

Start with `SKILL.md`.

Key bundled resources:
- `scripts/profile_tabular_data.py` — profile CSV/TSV/XLSX input and recommend chart families
- `scripts/find_samples.py` — find the closest plotting templates in this repository
- `references/data-to-chart-guide.md` — chart selection heuristics
- `references/selection-rubric.md` — template selection heuristics
