# figures

Selected reference images used by the sample library.

## Location

All selected figures live in:

```text
ref/figures/selected/
```

The repository no longer separates image-origin and PDF-origin figures into different folders.
If a figure is selected for the library, it belongs in the same place.

## Naming

Figure filenames stay sample-centric:
- `sample_0006_flame_coordinate_vs_flow.jpg`
- `sample_0013_reduced_order_temperature_solutions.jpg`

That keeps the visual reference easy to trace back to the sample that uses it.
